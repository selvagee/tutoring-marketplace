### TeacherOn Local Deployment Scripts ###

This folder contains scripts to help with your local deployment of the TeacherOn application.

1. install_dependencies.bat - For Windows users to install required dependencies
2. start_app.bat - For Windows users to start the application
3. install_dependencies.sh - For Linux/Mac users to install required dependencies
4. start_app.sh - For Linux/Mac users to start the application

Please refer to LOCAL_DEPLOYMENT_INSTRUCTIONS.md for complete deployment instructions.
